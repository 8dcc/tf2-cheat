The cheat currently has the following features:
- ESP
  - Player ESP
    - Toggle friends only/enemies only/all
    - Box
    - Skeleton
    - Health
    - Name
    - Player class
    - Weapon
  - Building ESP
    - Toggle friends only/enemies only/all
    - Toggle building types
    - Building box
    - Building health
    - Building name (type of building)
  - Ammo ESP
  - Custom colors
- Aimbot
  - Custom FOV
  - Smoothing
  - Choose hitboxes
  - Silent
  - Disable if spectated in 1st person
- BunnyHop
- Autostrafe
  - Legit
  - Rage
- Spy autostab
- Spectator list
- Watermark
